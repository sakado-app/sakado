## Building

### Requirements

- Node.JS
- Ionic SDK (`npm install -g ionic`)
- Cordova (`npm install -g cordova`)
- Android SDK (Android)
- XCode (iOS)

### Android

```
$ ionic cordova build android
```

### iOS

Open platforms/ios on XCode and build the app from it
